## Лабораторная работа 5 - Agile и Scrum

**Тема проекта** - каталог фильмов и сериалов

Веб-приложение для каталогизации фильмов и сериалов

Пользователи могут просматривать каталог, искать фильмы по названию, 
сортировать по рейтингу или дате выхода, добавлять в избранное

Приложение поддерживает CRUD-операции для администраторов

## Sprint Review

### Задачи, закрытые в спринте:
1. **Создать главную страницу с каталогом** - разработана и реализована главная страница приложения с отображением каталога фильмов. Пользователи могут видеть карточки фильмов с постерами и рейтингами

2. **Реализовать добавление в избранное** - создана система избранного. Пользователи могут добавлять фильмы в избранное, просматривать свой список. Данные сохраняются в localStorage

### Что было доставлено по итогу спринта:
- Рабочая главная страница каталога фильмов с адаптивным дизайном
- Функционал "Избранного" с сохранением состояния
- Базовый интерфейс приложения готов к использованию
- Пользователи могут просматривать каталог и сохранять понравившиеся фильмы

### Задачи, не завершённые в спринте:
1. **Реализовать поиск фильмов** - в процессе разработки. Реализован поиск по названию, осталось добавить поиск по жанрам и актерам

2. **Создать страницу деталей фильма** - в процессе. Готов макет и верстка, осталось подключить API для получения данных и добавить трейлеры

3. **Добавить сортировку каталога** - Не начата. Перенесена на следующий спринт из-за нехватки времени

## Sprint Retrospective

### Что прошло хорошо:
- Успешно создана главная страница
- Быстро реализован функционал избранного с сохранением состояния

### Что пошло не так:
- Серьезно недооценена сложность реализации поиска (8 story points оказалось мало)
- Задержались с интеграцией API для страницы деталей фильма
- Не учтено время на написание unit-тестов для функционала избранного
